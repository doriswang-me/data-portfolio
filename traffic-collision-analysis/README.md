# California Traffic Collision Analysis (2020–2025)

## Overview
This project analyzes over **2.46 million traffic collision records** in California to identify regional and temporal patterns in crash severity.  
Rather than relying solely on raw collision counts, the analysis introduces a **severity scoring framework** to better assess the real-world impact of crashes and support data-driven safety insights.

## Objectives
- Identify trends in crash severity across time and geographic regions
- Move beyond collision frequency by incorporating severity-based evaluation
- Translate contributing factors into actionable safety and risk insights

## Data
- California traffic collision data (2020–2025)
- Over 2.46M records with information on location, time, weather, speed, and impairment

## Methodology
- Data cleaning and preprocessing using **Python (Pandas)**
- Exploratory data analysis to uncover temporal and regional trends
- Construction of a **severity scoring framework** to evaluate crash impact
- Analysis of contributing factors including weather conditions, speeding, and impairment

## Key Insights
- Crash severity varies significantly by region and time period
- Severity-based analysis reveals risks that are not visible through raw collision counts
- Weather, speeding, and impairment play a critical role in high-impact crashes

## Tools
- Python
- Pandas
- Data Visualization libraries

## Deliverables
- Cleaned and structured datasets
- Analytical notebooks and scripts
- Visualizations highlighting severity trends and risk factors
